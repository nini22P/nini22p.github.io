## 安装方法

### 实机

1. 在实机内安装原版游戏和 repatch 插件
2. 下载汉化补丁并解压
3. 将解压得到的 repatch 文件夹复制到 ux0 文件夹内

### 模拟器

1. 在模拟器内安装原版游戏
2. 下载汉化补丁
3. 在模拟器菜单栏中依次选择 `文件` -> `安装zip、vpk` -> `选择文件`，然后选择下载的汉化补丁
4. 等待安装完毕

## 安装方法

### PSV 实机

1. 确保实机已安装原版游戏及 **repatch** 插件。
2. 下载汉化补丁并解压。
3. 将解压得到的 `repatch` 文件夹通过 FTP 或 USB 复制到 `ux0:/` 目录下。

### Vita3K 模拟器

1. 在模拟器内安装原版游戏。
2. 下载汉化补丁压缩包。
3. 在模拟器菜单栏中依次选择 `文件` -> `安装 .zip、.vpk` -> `选择文件`。
4. 选择下载的补丁包并等待安装完成。
